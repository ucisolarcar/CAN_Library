### Requirements
- Linux (or WSL https://code.visualstudio.com/docs/remote/wsl)
- cmake
- make

### Build
``` bash
    ./build.sh
```

### Clean
``` bash
    ./clean.sh
```

### Testing 
- write your test in the test_main.cpp file
- build the program using the build script
``` bash
    ./test.sh
```
